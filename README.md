# 🎮 Par ou Ímpar em Java

Um simples jogo de **Par ou Ímpar** desenvolvido em Java para praticar conceitos básicos de programação, como entrada de dados, números aleatórios, arrays e estruturas condicionais.

## 📖 Sobre o Projeto

Neste jogo, o jogador escolhe entre **Par** ou **Ímpar** e informa um número entre 1 e 10. O computador gera um número aleatório no mesmo intervalo, e a soma dos dois números determina o vencedor.

## 🚀 Funcionalidades

* Escolha entre Par ou Ímpar.
* Entrada de números pelo teclado.
* Geração automática de números aleatórios.
* Exibição do resultado da soma.
* Verificação automática do vencedor.

## 🛠️ Tecnologias Utilizadas

* Java
* Scanner (entrada de dados)
* Random (geração de números aleatórios)

## ▶️ Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/Sebasduhg/par-ou-impar-java.git
```

### 2. Entre na pasta do projeto

```bash
cd par-ou-impar-java
```

### 3. Compile o código

```bash
javac Main.java
```

### 4. Execute o programa

```bash
java Main
```

## 🎲 Exemplo de Execução

```text
Escolha a sua opção:
(0) para IMPAR
(1) para PAR

Soma da jogada: 7 + 4 = 11
```
